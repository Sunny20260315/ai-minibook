Openclaw 对 mac 更友好，以下所有操作都是基于 mac 的。

一、前置基础环境

检查前置条件
打开 终端 (Terminal)，依次执行：

```
# 检查是否安装了 Node.js (需要 v22 以上)

node -v

# 检查是否安装了 git

git --version
```

- 如果没有 Node.js：去 nodejs.org 下载 LTS 版安装。
- 如果没有 git：用 Homebrew 安装 brew install git，或者直接下载 Xcode Command Line Tools。

# 二、 一键安装 OpenClaw (官方推荐方式)

## 通过 npm 全局安装

```
npm install -g openclaw
```

安装完成后验证：

```
openclaw --version
```

## 初始化配置 (创建你的第一个 Gateway)

```
openclaw onboard --install-daemon
```
